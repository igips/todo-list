# Todo List

Todo list app made with HTML, CSS, JS and FIREBASE.

[Live Demo](https://igips.github.io/todo-list/) :point_left:

## Features
* Responsive design
* Sign Up/In 
* Add/Delete projects
* Add/Edit/Delete tasks
* Mark tasks as completed
* Sort tasks by date

## Built with
- [Firebase](https://firebase.google.com/)
